🗺️ 확장 지도 모달 테스트 - 모달 교체 방식

📋 테스트 가이드

• 모달 교체 방식: 상세 모달 → 지도 모달 → 상세 모달로 자연스럽게 전환

• 뒤로가기: 지도 모달에서 "← 상세 정보로" 버튼 클릭

• 완전 닫기: 지도 모달에서 "완전 닫기" 버튼 클릭

• 키보드: ESC 키로 뒤로가기, 외부 클릭으로 뒤로가기

• 애니메이션: 부드러운 fade-out/fade-in 전환 효과

🔧 테스트 시나리오

1️⃣

기본 플로우 테스트

버튼 클릭 → 지도 모달 열림 → 뒤로가기 → 상세 모달 복귀

2️⃣

미니맵 클릭 테스트

미니맵 클릭 → 지도 모달 열림 → 뒤로가기 → 상세 모달 복귀

3️⃣

키보드 접근성 테스트

Tab으로 포커스 → Enter/Space로 열기 → ESC로 뒤로가기

4️⃣

완전 닫기 테스트

지도 모달에서 "완전 닫기" → 모든 모달 닫힘

5️⃣

외부 클릭 테스트

지도 모달 외부 클릭 → 뒤로가기 동작
